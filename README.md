# CDL Auto Stats Analyser

This version does **not** need BetMGM or manual odds input.

It pulls public CDL / Breaking Point pages and ranks:

- Best players from current stats
- Team strength
- Upcoming matches
- Stronger likely team based on team stats

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Phone usage

Run it on a PC/laptop or cloud host, then open the Streamlit network URL on your phone.

## Notes

This is an analysis dashboard. It does not place bets and does not guarantee profit.

The public pages may change layout. If that happens, use the Debug tab to see what the app is reading.
