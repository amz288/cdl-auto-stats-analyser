# CDL Auto Stats Analyser v2

This version is more robust than v1.

It pulls Breaking Point team pages, extracts rosters/player stats where available, ranks players, builds a team strength table, and parses upcoming matches.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

Upload these files to GitHub:

- app.py
- requirements.txt
- README.md

Then deploy with Streamlit Community Cloud using:

- Branch: main
- Main file path: app.py
